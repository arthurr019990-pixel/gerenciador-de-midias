# gerenciador-de-midias
Esse é um projeto progaramado em python cujo objetivo é servir como um gerenciador de midias, onde o usuário pode inserir filmes, revistas ou livros os quais deseja guardar com o objetivo de ler/ver posteriormente, como se fosse um diário. O projeto conta com uma interface GUI e um banco de dados com funcionalidades CRUD.

# Execução da aplicação
Para executar a aplicação basta abrir o editor de código da sua escolha e executar o arquivo gui_midias.py
